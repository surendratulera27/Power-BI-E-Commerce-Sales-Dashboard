# Power-BI-E-Commerce-Sales-Dashboard
The E-Commerce Sales Dashboard is an interactive Power BI report designed to analyze and visualize sales performance. The dashboard helps stakeholders track revenue, profit, quantity sold and order placed that enabling data-driven business decisions.

Dataset "Details.csv" & "Orders.csv" linked by common column "Order_ID"

Dashboard : Showcasing multiple charts & graphs in a single view. It helps to analysis any business in complete 360 view.
Charts/Graphs :
1.Profit vs Month(Order Date) (column chart) : to showcase monthly profit.
2.Sub-Category vs Amount (bar chart)
3.PaymentMode vs Order ID (donut chart)
4.Quantity vs Category (pie chart) : to checkout how much stock is sold out in each category(electronics, furniture, etc)
5.City vs Amount (bar cahrt) : total revenue generated from top cities.

Key KPIs : to showcase the most important information.
Total Revenue
Total Profit
Total Quantity Sold
Total Order Placed

Slicers(State, Quarterly) : to showcase any choose/filter option.
Title : textbox
